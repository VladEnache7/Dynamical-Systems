### üìö Key Topics Breakdown

<details>
<summary><strong>Click to expand detailed topic list</strong></summary>

#### Part I: Continuous Dynamical Systems (Differential Equations)
*   **Linear Equations:** Constant coefficients, Homogeneous vs. Non-homogeneous.
*   **Methods:** Lagrange method (Variation of Parameters), Method of Undetermined Coefficients.
*   **Qualitative Analysis:**
    *   **Phase Portraits:** Visualizing trajectories without solving explicitly.
    *   **Stability:** Stable, Unstable, Asymptotically Stable equilibria.
    *   **Linearization:** Using the Jacobian matrix to analyze nonlinear systems.

#### Part II: Discrete Dynamical Systems (Difference Equations)
*   **Recurrences:** Linear recurrences (e.g., Fibonacci).
*   **Maps:** The Logistic Map, Fixed points, Chaos.
*   **Numerical Analysis:**
    *   **Euler's Method:** First-order numerical procedure.
    *   **Runge-Kutta:** More accurate approximation for DEs.
